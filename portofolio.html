About Me

Nama saya Muhammad Huseini. Saya kuliah di Universitas Malikussaleh, jurusan S1 Teknik Elektro. Saat website ini dibuat saya masih menjalani semester 6. Saya tinggal dimedan, saya anak pertama dari 5 bersaudara, ayah saya bekerja sebagai pns dan ibu saya bekerja sebagai dokter tapi untuk saat ini tidak bekerja dikarnakan sedang sakit keras, dan saya merantau keaceh untuk menimba ilmu dikampus tercinta saya yaitu, Universitas Malikussaleh.

Saya aktif dalam organisasi baik internal maupun eksternal, organisasi internal yang saya ikuti yaitu DPM (Dewan Perwakilan Mahasiswa) dan organisasi eksternal yang saya ikuti yaitu IMKM (ikatan Mahasiswa Kota Medan). Mengenai hobi, saya memiliki beberapa hobi diantaranya bermain badminton,berenang,bersih-bersih,membaca komik dll.

Keseharian saya saat dikos tidak jauh berbeda dengan kebanyakan anak kos lainnya, pada pagi hari saya bangun langsung scroll tiktok,kalau sudah bosan saya membuat kopi dan minum didepan halaman sambil menghisap roko surya (FYI :jadwal kelas saya selalu saya buat siang atau menjelang siang), pada siang hari biasanya saya nasuk kelas tapi kebanyakan titip absen, dan pada sore harinya saya baru mencari makan untuk mengisi kekosongan perut saya.
